# MyHabit

## アプリ概要
ダイエット習慣化を目的に、体重記録と習慣トラッカーの機能を併せ持つアプリです。
日々の体重推移と自身が設定したダイエット習慣の達成度をグラフで可視化することによって、継続のモチベーションを維持する設計にしました。

## アプリ作成動機
テーマを決定した理由は、自分自身や周囲の友人が抱える「ダイエットを継続できない」という課題を解決するためです。継続できない一番の原因を周囲の友人に聞いたところ、ダイエットのために頑張った行動と結果が結びついている感覚がないことが要因であると発覚しました。したがって、ダイエット継続に必要な要素は、体重推移と日々の努力が紐づく感覚をユーザーが得ることだと考えました。

## 差別化ポイント
既存のダイエット系アプリの特徴は、主に以下の2点です。
- 体重記録アプリか習慣トラッカーアプリに二極化されており、両方の機能を兼ね備えたアプリは少ない
- 習慣化したいことがプリセットされており、ユーザーオリジナルの習慣登録ができない

上記を踏まえて、本アプリは以下の機能に決定しました。  
- 体重・体脂肪率・モチベーション記録 + 習慣トラッカー
- ユーザーがオリジナルで習慣を設定
- 体重推移と日々の習慣達成率のグラフによる可視化
